撩课学院（itlike .com）起源于一个资深的教育团队所发起的一个公益项目。从2017年开始，通过各大在线平台(网易、腾讯等)，向全国免费开放他们精心打磨的视频课程 “h5系列” 、 “python系列” 、 “java系列” 等。在短短不到一年的时间， 接近10万多名学生报名参加学习了这些课程, 好评如潮。为了有组织的推出更加全面、优质、成体系的课程，撩课学院应运而生。撩课是一个对课程质量追求极致的教学团队，保证交到每位学员手中的都是一件完美艺术品。
